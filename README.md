# stylelint-config

We Make Website's stylelint rules and config.

## Installation

Run the command:

```
yarn add --dev stylelint @we-make-websites/stylelint-config
```

## Usage
We Make Websiteâ€™s stylelint rules come bundled in `@we-make-websites/stylelint-config`. To enable these rules add the following to your _.stylelintrc root file:

```js
extends: [
  '@we-make-websites/stylelint-config',
],
```
